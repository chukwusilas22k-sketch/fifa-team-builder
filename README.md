# fifa-team-builder
Python application that generates soccer team formations and assigns players based on position and attributes.

This project is a Python application that generates soccer team formations
based on player roles and attributes.

## Features
- Generates optimized soccer lineups
- Assigns players to positions automatically
- Supports multiple formations
- Stores player data using SQLite

## Technologies
- Python
- SQLite

## Example Use
The program allows users to input player attributes and automatically generate
balanced teams based on formation rules.
